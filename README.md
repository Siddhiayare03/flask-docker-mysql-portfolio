# Flask Portfolio Website - DevOps Project

## Project Overview

A Flask Portfolio website deployed using Docker, Docker Compose,
Jenkins CI/CD, GitHub and AWS EC2 with MySQL database integration.

## Tech Stack

- Python
- Flask
- MySQL
- Docker
- Docker Compose
- Jenkins
- Git
- GitHub
- AWS EC2
- Linux (Ubuntu)

## Features

- Portfolio Website
- Contact Form
- MySQL Database
- Dockerized Application
- CI/CD Pipeline using Jenkins
- Cloud Deployment on AWS EC2
